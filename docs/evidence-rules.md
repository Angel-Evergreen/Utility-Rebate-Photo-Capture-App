# Evidence Rules

## Rule Evaluation Flow
1. User selects rebate program
2. User selects services performed
3. System loads required evidence for each service
4. System generates a checklist
5. Each checklist item must have at least one photo
6. Job cannot be marked "Complete" until all required evidence exists

## Enforcement Rules
- Photos must be captured inside the app
- Photos are automatically timestamped
- Photos cannot be uploaded without a linked requirement
