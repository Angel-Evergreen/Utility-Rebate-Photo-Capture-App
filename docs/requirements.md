# Functional Requirements

## FR-1 Guided Intake
Field crew must complete the following intake fields before photo capture:
- Crew member name
- Customer name
- City

## FR-2 Required Fields Enforcement
The application must prevent photo capture and submission unless all required intake fields are completed.

## FR-3 Photo Capture
Photos must be taken using the in-app camera.

## FR-4 Metadata
Each photo submission must automatically include:
- Crew member name
- Customer name
- City
- Date
- Time

## FR-5 Storage
Photos and associated metadata must be securely stored in cloud storage and retrievable by office users.

## FR-6 Access Control
- Field crew users can submit photos
- Office users can view and download submissions
