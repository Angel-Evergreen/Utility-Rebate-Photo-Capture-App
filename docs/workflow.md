# User Workflow

## Field Crew Workflow
1. Crew member opens the mobile app
2. App prompts the crew member to enter:
   - Crew member name
   - Customer name
   - City
3. Crew confirms intake details
4. Crew takes required photos using the in-app camera
5. Photos are automatically timestamped
6. Crew submits photos
7. Confirmation screen is displayed

## Office Workflow
1. Office accesses the admin dashboard
2. Office filters submissions by:
   - Customer name
   - City
   - Date
3. Office reviews and downloads photos
4. Photos are used for PSE compliance submissions
